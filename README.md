# Felicina_schedule
Приложение для формирования списка мероприятий для музея им филицина<br />
Ссылка на гугл таблицу https://docs.google.com/spreadsheets/d/1s41VA87rQ_1kecpIxGRmQ3Q2wegeGdXOyCLXy3tnfew/edit?gid=0#gid=0<br />
Для форматирования заголовков следует поставить символ "@" где необходимо сделать перенос на следующую строку.<br />
Чтобы мероприятие не отображалось, следует поставть "1" в поле "hidden"<br />
Сервис API https://app.sheetbest.com
